# 문서 텍스트 추출 기능 안내

## 개요
이 기능은 사용자가 업로드한 문서 파일 (PDF, DOCX, TXT 등) 의 내용을 자동으로 추출하여 AI 가 분석할 수 있도록 합니다.

## 지원되는 파일 형식

### ✅ 완전히 지원
- **PDF 파일** (.pdf) - pdf.js 라이브러리를 사용하여 모든 페이지의 텍스트 추출
- **Word 문서** (.docx) - mammoth.js 라이브러리를 사용하여 텍스트 추출
- **텍스트 파일** (.txt, .csv) - 직접 텍스트 읽기
- **HTML 파일** (.html, .htm) - HTML 태그 제거 후 텍스트 추출

### ⚠️ 기본 지원 (메타정보만)
- **Excel 파일** (.xls, .xlsx) - 파일 이름과 크기 정보만 제공 (완전한 추출을 위해서는 SheetJS 라이브러리 추가 필요)
- **PowerPoint 파일** (.ppt, .pptx) - 파일 이름과 크기 정보만 제공 (전문 라이브러리 추가 필요)

## 작동 방식

### 1. 파일 첨부
사용자가 파일 첨부 버튼 (📎) 으로 문서를 선택하면:
- 이미지는 기존 방식대로 Dify 에 업로드
- 문서는 클라이언트에서 텍스트 추출

### 2. 텍스트 추출
- 파일 형식에 따라 적절한 라이브러리 사용
- 추출된 텍스트는 캐시에 저장 (동일 파일 재사용 시 성능 향상)
- 추출 중에는 "문서 내용 추출 중..." 표시

### 3. AI 에 전송
- 추출된 텍스트가 사용자 질문 뒤에 자동 추가
- 형식: `[사용자 질문] + [문서 내용]`
- AI 는 문서 내용을 기반으로 답변 생성

### 4. 결과 표시
- 채팅 화면에 첨부 파일 아이콘과 파일명 표시
- 파일 형식별 고유 아이콘 사용 (PDF, Word, Excel 등)

## 기술 스택

### 외부 라이브러리
```html
<!-- PDF 텍스트 추출 -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js"></script>

<!-- Word 문서 텍스트 추출 -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/mammoth/1.6.0/mammoth.browser.min.js"></script>
```

### 주요 함수
- `extractTextFromDocument(file)` - 파일 형식 감지 및 텍스트 추출
- `extractTextFromPDF(file)` - PDF 텍스트 추출
- `extractTextFromDOCX(file)` - DOCX 텍스트 추출
- `extractAllAttachmentsText(attachments)` - 여러 파일의 텍스트 일괄 추출
- `getFileIcon(fileName)` - 파일 형식별 아이콘 반환

## 사용 예시

### 시나리오 1: PDF 보고서 분석
1. 사용자가 "fileAttachBtn" 클릭
2. PDF 파일 선택 (예: "monthly_report.pdf")
3. 질문 입력: "이번 달 매출 주요 이슈를 요약해줘"
4. 시스템 자동 동작:
   - PDF 에서 텍스트 추출
   - 질문에 추출된 텍스트 추가
   - AI 에 전송
5. AI 가 보고서 내용을 분석하여 답변

### 시나리오 2: 여러 문서 동시 분석
1. 여러 파일 첨부 (PDF + DOCX + TXT)
2. 질문: "첨부한 문서들에서 공통된 문제를 찾아줘"
3. 시스템: 모든 문서에서 텍스트 추출 후 통합
4. AI 가 여러 문서를 cross-reference 하여 답변

## 성능 최적화

### 캐싱
- 추출된 텍스트는 `state.extractedTextCache` 에 저장
- 동일한 파일을 재사용할 경우 재추출 방지
- 세션 종료 시 캐시 초기화

### 비동기 처리
- 모든 추출 작업은 `async/await` 사용
- UI 블로킹 방지
- 진행 상황 표시

### 에러 처리
- 각 추출 함수는 try-catch 로 감싸짐
- 추출 실패 시에도 AI 요청 계속 진행
- 오류 메시지 표시

## 한계 및 향후 개선

### 현재 한계
1. **Excel/PPT 완전 추출 불가** - 메타정보만 제공
2. **이미지 내 텍스트 추출 불가** - OCR 필요
3. **복잡한 포맷팅 손실** - 순수 텍스트만 추출
4. **파일 크기 제한** - 브라우저 메모리 제한

### 향후 개선 사항
1. **SheetJS 라이브러리 추가** - Excel 완전 추출
2. **Tesseract.js 추가** - OCR 기능 (이미지 내 텍스트)
3. **html-to-text 라이브러리** - 더 정교한 HTML 파싱
4. **스트리밍 추출** - 대용량 파일 지원
5. **추출된 텍스트 편집 기능** - 사용자가 수정 가능

###常见问题排查
1. **라이브러리 로드 실패**: CDN 연결 확인
2. **PDF 추출 실패**: 파일 손상 여부 확인
3. **DOCX 추출 실패**: 암호화된 파일인지 확인
4. **추출된 텍스트가 비어있음**: 파일 형식이나 내용 확인

## 보안 고려사항

- 모든 처리는 클라이언트에서 수행 (서버 업로드 없음)
- 추출된 텍스트는 AI API 에만 전송
- 캐시는 메모리에만 저장 (세션 종료 시 소멸)
- 파일 원본은 변경되지 않음
